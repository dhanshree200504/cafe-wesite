# The Stone Age Cafe — Nerul, Navi Mumbai

An elevated, cave-inspired digital experience for **The Stone Age Cafe**, a 100% pure vegetarian Continental restaurant located in the heart of Nerul.

---

## ✦ About the Project
This is a high-performance, single-page web application designed to showcase the unique rustic ambiance and extensive menu of the cafe.

* **100% Pure Vegetarian**: Highlights a diverse menu of 70+ items.
* **Cave-Themed Design**: Features a custom UI that reflects the cafe's rustic stone walls and amber lighting.
* **Interactive Menu**: A dynamic, category-based menu with smooth-scroll navigation and a guest-favorite highlight ticker.
* **Smart Reservations**: Integrated WhatsApp booking system that generates pre-filled reservation messages for instant confirmation.

## 🍝 Featured Menu Highlights
* **Signature Pasta**: Black Garlic Pasta & Seven Pepper Bae.
* **Gourmet Burgers**: The Barbec-you-3000 signature burger.
* **Guest Favorite**: Legendary Hazelnut Cold Coffee.
* **Board Games**: A dedicated space for social gaming and comfort.

## 🛠️ Technical Stack
* **Frontend**: HTML5, CSS3 (Custom animations & responsive layout).
* **Logic**: Vanilla JavaScript for dynamic menu rendering and reservation handling.
* **Integrations**: Direct links for Zomato, Swiggy, and EazyDiner.
